# Opis projekta

Ta projekt predstavlja preprosto spletno stran z osnovnimi funkcionalnostmi.

## Glavne funkcionalnosti

1. **Registracija uporabnika** – uporabnik lahko ustvari svoj račun.
2. **Prijava uporabnika** – preverjanje pristnosti uporabnika.
3. **Uporabniški profil** – prikaz osnovnih podatkov o uporabniku.
4. **Kontaktni obrazec** – uporabnik lahko pošlje sporočilo administratorju.
5. **Sistem objav** – prikaz objavljenih vsebin.
